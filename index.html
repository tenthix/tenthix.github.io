MEET TENTHIX

LIGHT-FAST TRANSPARENT TRANSACTIONS

Tenthix protocol is designed to transfer transactions across network is JSON format that is essential for light-fast and transparency of transactions.

NO MORE MINING OR STAKING

Tenthix uses inovative lightweight transaction verification algorythm called STARLIGHT that is based on simple mathematic calculations and real-time database checking that does not require staking or mining.

SCALABILITY

Tenthix network is highly scalable network that can be implemented or used as core in variety of project and platforms that require lightweight and fast transaction system.

TECHNOLOGY OF TENTHIX NETWORK

PEER

CONNECTING NODES TOGETHER

Peers are core layer of network that is simply providing essential information for nodes to keep network properly synchronized.

WALLET

MANAGES TIXC COINS

Wallet is end-users tool that provides acess to TIXC coins on network using unique security tokens.
"One key, One vault"

NODE

NETWORK MAINTAINER

Nodes are top layer of network that is processing and validating transaction and keeps database synchronized across whole network.
